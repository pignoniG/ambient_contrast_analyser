  • Display white point luminance cd/m2: Luminous flux per unit solid angle per unit projected source area. (cd/m2 or Nit) Often referred as Brightness in displays specifications.

  • Display static contrast ratio: ratio of the luminance of the brightest color (white) to that of the darkest color (black) that the system is capable of producing at the same time.

  • Display black point luminance cd/m2: luminance of the darkest color (black) that the system is capable of producing. The reference veiling glare luminance specified the IEC 61966-2-1 document is 0.2 cd/m2 (this is the reference viewer-observed black point luminance) and can be used if contrast ratio or display black point are unknown.

  • Display Reflectance: Normalized value (0-1) of the proportion of ambient light reflecting off the display. Normal values can be 0.1 for a matte display or as low as 0.047 for a treated glossy display (iPhone 7).
